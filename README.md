# 🎯 IntenTube

Your Goals. Not The Algorithm.

IntenTube helps students and professionals use YouTube intentionally by reducing distractions and blocking non-productive content.

## Features

- 🚫 Homepage distraction blocking
- 🎯 Goal tracking
- 📊 Focus analytics
- ⏱ Time saved tracking
- 🔥 Focus streaks
- 📚 Educational keyword filtering
- 🧠 Smart recommendation blocking


## Installation

### For Regular Users (Recommended)
1. Go to the **Releases** page of this repository (on the right side).
2. Download the latest `StudyTube-Release.zip` file.
3. Extract the downloaded ZIP file to a folder on your computer.
4. Open Google Chrome and navigate to `chrome://extensions/`.
5. Enable **Developer mode** using the toggle in the top right corner.
6. Click the **Load unpacked** button in the top left corner.
7. Select the folder you extracted the ZIP into.
